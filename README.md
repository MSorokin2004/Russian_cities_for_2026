# Города России на 2026 год | Russian cities for 2026

Собрал города из таблички со страницы рувики https://ru.ruwiki.ru/wiki/Список_городов_России?utm_referrer=https://www.google.com/

Всего 1248 городов, 15 из которых федерального значения (ДНР, ЛНР, Херсонская и Запорожская области присутствуют!).

Поля таблицы: 
  - Номер;
  - Название города;
  - Название Региона (Республики названы типа Крым, Якутия, Коми, Башкирия и тд.);
  - Федеральный округ;
  - Население по переписи 2021 года (для новых территорий пропуск: ДНР, ЛНР, Херсонская и Запорожская области);
  - Основание / первое упоминание (вместо веков пропущенные значения, так как поля числовые);
  - Статус города (то есть когда был присвоен официально);
  - Ссылка на статью РуВики с городом;

Вторая таблица содержит только город, регион, округ.

Перед использвоанием рекомендую проанализировать данные, хотя бы взглядом пробежаться.

Добавить номер региона к значениям проще всего через функцию ВПР в Excel и таблицу регионов, но предварительно надо будет привести наименования регионов к стандартному виду.
